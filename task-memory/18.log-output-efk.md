# 系統 Log 資料收集至 EFK

今晚簡略說一下，如何透過HELM安裝EFK在K8s中

## EFK 安裝

```bash
# install EFK
helm install --name efk stable/elastic-stack -f efk-values.yaml

# upgrade
helm upgrade efk stable/elastic-stack -f efk-values.yaml

```

基本上以下幾項，勢必要安裝之項目，其他則是需求而選

* elasticsearch
